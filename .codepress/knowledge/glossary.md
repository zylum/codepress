# Glossary

Terminology used in this codebase that might be unclear.

> Seed this file using `skills/bootstrap.md` or fill in manually.
> Include: domain-specific terms, abbreviations, internal naming conventions.

---

| Term | Definition |
|---|---|---|
| tokens-in | Input/prompt tokens consumed by a model call for a slug. Recorded in cost signals. |
| tokens-out | Output/completion tokens generated by a model call for a slug. Recorded in cost signals. |
| cost-estimate | Estimated USD cost of model usage for a slug. Self-reported or wrapper-captured. |
| self-reported | Cost data captured by the builder estimating their own token usage. V0 default. |
| budget variance | Difference between budgeted and actual cost/tokens for a galley. Computed at review and by the cost-check gate. |
| cost-check | A verify gate that reads galley budget and compares against aggregated cost signals. Defined in codepress.md. |
