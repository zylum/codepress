# CodePress

A compound product development operating system for the AI era.

**On this page:** [What it is](#what-it-is) · [Three pillars](#three-pillars) · [The loop](#the-loop) · [Quick links](#quick-links)

---

![CodePress overview](overview.jpeg)

## What it is

CodePress is a structured approach to building products with AI.
Every initiative is shaped before it is built.
Every delivery leaves behind reusable knowledge.

```
Idea → Shape → Build → Learn → Compound
```

The software is the output. The growing capability of the system is the asset.

---

## Three pillars

### System — How we work
Playbooks, skills, templates, hooks, and configuration that define the operating model.

### Product — What we build
Initiatives, galleys, slugs, and releases — the product factory.

### Knowledge — What we know
Concepts, glossary, decisions, patterns, and current state — shared understanding.

A **Compound Loop** runs across all three, continuously improving them.

---

## The loop

```
Signal → Pattern → Knowledge
```

Raw observations from delivery become reusable lessons that update shared understanding.
Every cycle leaves the organisation smarter.
